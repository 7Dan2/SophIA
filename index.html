Aide et informations

Qu'est ce que SophI.A ?

SophI.A est un générateur de nommage de fichier pour les CE

Comment cela fonctionne ?

Dans la partie gauche, l'utilisateur entre les informations dans les champs prévus, il est guidé par un code couleur vert, orange et rouge, ainsi que des fenêtres d'alerte.

Chaque champ de saisie comporte une ou plusieurs vérification :

  • Si la valeur atendue est correcte, le fond du champ devient vert
  • Si la valeur attendue est incorrecte, le fond devient rouge
  • Si la valeur attendue est partiellement correcte, le fond devient orange et le texte rouge
  • Si la valeur attendue est incorrecte mais n'a qu'une valeur informative, le fond devient orange
  • Dans certain cas c'est la valeur saisie qui s'affiche en orange sur fond vert pour attirer l'attention de l'utilisateur.

A noter que les erreurs bloquent la génération des noms

Une fois que l'utilisateur à terminé sa saisie et qu'aucune erreur n'est présente, l'appui sur le bouton 'Générer' anime la partie droite, où s'affichent les noms de fichiers, avec l'extension .pdf
L'utilisateur peut surligner à sa convenance soit le nom seul ou la totalité avec l'extension pour ensuite renommer ses fichiers.
